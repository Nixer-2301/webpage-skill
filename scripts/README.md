# 字体资源设置指南

本技能建议使用以下免费字体以获得最佳视觉效果，但这些字体**不包含在仓库中**，需要您手动下载并放置到 `assets/fonts/` 目录。

## 手动下载步骤

1. **MiSans**  
   访问 https://hyperos.mi.com/font ，点击下载按钮，解压后将 `MiSans` 文件夹放入 `assets/fonts/`。

2. **阿里巴巴普惠体 3.0**  
   访问 https://fonts.alibabagroup.com ，选择“阿里巴巴普惠体3.0”下载，解压后将 `AlibabaPuHuiTi-3` 文件夹放入 `assets/fonts/`。

3. **HarmonyOS Sans**  
   访问 https://developer.huawei.com/consumer/cn/design/resource/ ，下载 `HarmonyOS Sans` 压缩包，解压后将文件夹放入 `assets/fonts/`。

完成后，目录结构应类似：
assets/fonts/
├── MiSans/
├── AlibabaPuHuiTi-3/
└── HarmonyOS Sans/

## 自动脚本（实验性）

仓库曾包含自动下载脚本，但因官方链接经常变更且需接受许可协议，已被移除。请务必手动下载以确保合规。
